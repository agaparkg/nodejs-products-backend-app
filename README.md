/ - Home page  
GET - /products - All products  
GET - /products/:id - Single product  
POST - /products (req.body) - Add new product  
PATCH - /products/:id (req.body & req.params.id) - Update a product  
DELETE - /products/:id (req.params.id) - Delete a product
